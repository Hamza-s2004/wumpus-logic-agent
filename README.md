# Wumpus Logic Agent

A web-based AI agent that uses Propositional Logic, CNF, and Resolution to safely navigate a Wumpus world.

## Features
- Dynamic grid
- Percepts (Breeze, Stench)
- Knowledge Base (KB)
- CNF Conversion
- Resolution-based inference
- Interactive UI

## Run
1. pip install flask flask-cors
2. python logic.py
3. open index.html
